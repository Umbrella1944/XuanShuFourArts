# XuanShuFourArts

Local source project for the Steam Workshop mod **Four Arts of the Mystic Pivot**.

## Current Release

- Mod version: `1.0.0.8`
- Supported game version: `1.0.13`
- Workshop FileId: `3747599301`
- Author: `Umbrella`

## Output

Release files are generated into `dist` and synced to:

`C:\Program Files (x86)\Steam\steamapps\common\The Scroll Of Taiwu\Mod\XuanShuFourArts`

Important files:

- `dist\Config.Lua`
- `dist\Settings.Lua`
- `dist\Plugins\XuanShuFourArtsB.dll`
- `dist\Plugins\XuanShuFourArtsF.dll`
- `dist\xuanshu_sijue_character_cover2_1024.jpg`

## Current Feature Set

- Reworks four base sectless Tier 9 martial arts into Divine Tier 1 core arts.
- Keeps the original low-tier learning and usage requirements.
- Supports Chinese and English in-game text at game startup.
- Adds the four-piece set bonus when all four arts are equipped for Practice.
- Adds an in-game set tooltip with Alt details.
- Uses existing game effect systems where possible and keeps save interaction minimal.
